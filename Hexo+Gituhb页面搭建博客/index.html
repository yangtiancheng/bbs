搭建Github Page + Hexo博客系统

虽然网上有很多教程了，但是我还是想把自己的参考历程贴出来，给自己做一个参考

参考文档：使用Hexo+Github一步步搭建属于自己的博客（基础）

这里我只写简要的过程(Macbook Pro)

  1. 访问github创建一个新的博客仓库，仓库名称规格 XXX.github.io

  2. 提前安装好npm,使用npm安装hexo: npm install hexo -g

  3. 安装完成后，使用hexo -v检查是否安装成功

  4. 安装需要的组建 npm install

  5. 加载hexo: hexo g

  6. 启动访问服务: hexo s

  7. 通过http://localhost:4000访问本地Hexo博客
    (注意如果启动服务后，界面转圈，说明4000端口被占用，我们换4001端口尝试访问http://localhost:4001)

  8. 访问成功后，hexo绑定github page:

    git配置：
    git config –global user.name ‘Name’
    git config –global user.email 'Mail@163.com‘

    github 上传公钥(参考)

  9. 进入hexo的_config.yml文件绑定github page:
    滑动到文档末尾，添加：

    deploy:
    type: git
    repository: git@github.com:XXX/XXX.github.io.git(工程路径)
    branch: master

  10. 扩展一步上传博客内容到github仓库的组建：
    npm install hexo-deployer-git –save

  11. 创建博客：
    hexo new post “博客名称”

  12. 博客内容都在source/_post/目录中：
    同步更新到github page：
    hexo d -g

  13. 现在访问：
    https://github账户.github.io/博客工程名称/

    可能发现界面上的样式都失效了，因为我们没有定义_config的url：

    再次访问后成功。
