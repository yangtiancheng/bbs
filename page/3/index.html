Gitbook使用

使用说明文档

安装流程

  1. 首先在你的本地需要如下软件：
    Git ① 、Gitbook Editor ②

  2. 下载安装完成后，选择一个对应的目录，获取对应书籍:

    2.1 Windows获取方式：


      1. 选择一个适当的目录，鼠标点击右键，选择[]

      2. 输入：git clone https://rdc.hand-china.com/gitlab/13035/gitbook_test.git

    2.2 Mac获取方式：


      打开终端，选择一个合适的目录，比如 /User/用户名/XXX,在终端执行：

         1. cd /User/用户名/XX

         2. git clone https://rdc.hand-china.com/gitlab/13035/gitbook_test.git

         

安装过程中可能需要输入账户/密码(工号/handhand)
