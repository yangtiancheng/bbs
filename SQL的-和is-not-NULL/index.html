SQL的!=和is not NULL

!=

SQL的Where语句如果出现!=判断符，DB会将数据中type有值且不等于normal的记录查出，
但如果记录中对应的type字段是NULL的话，数据将不会被查出来。

is not NULL

将查出type非空的记录

这里需要特别注意筛选数据时，最好不要用!=，除非特别确认这个字段一定是必输字段。
